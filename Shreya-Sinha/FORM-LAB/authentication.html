STORES THE DATA IN KEY VALUE PAIRS SYNTAX OF LOCAL STORAGE: SAVE DATA: local storage.setItem('key','value'); RETRIEVE DATA: var value = localStorage.getItem('key'); ex: localStorage.setItem('username', 'john_doe'); var username = localStorage.getItem('username'); remove data: localStorage.removeItem('key'); clear stotrage: localStorage.clear(); delete all data from localstorage Why is json needed? why json is needed? we convert objects to strings using json.stringify() to store them in localstorage, and we can convert them back to objects using json.parse() when we retrieve them. This allows us to store complex data structures like arrays and objects in localstorage, which only supports string values. for example: { username:'john_doe' email:'abc@gmail.com' password:'12345' } JSON.stringify(): converts a JavaScript object or value to a JSON string, which can be stored in localStorage. ex: const user = {name : "Rahul"}; JSON.stringify(user); // '{"name":"Rahul"}' explanation: we have an object user with a property name. When we use JSON.stringify(user), it converts the object into a JSON string representation, which can be stored in localStorage. JSON.parse(): parses a JSON string and returns the corresponding JavaScript object or value, allowing us to retrieve and use the data stored in localStorage.
