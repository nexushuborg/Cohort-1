# MERN Cohort 1

Welcome to the MERN Stack Cohort 1 repository!

## Structure

Each student has their own folder where they will submit assignments.

## Workflow

1. **Fork** this repository
2. **Clone** your fork to your local machine
3. Create a **branch** named `yourname-weekX`
4. Work in **your folder only**
5. **Commit** your changes regularly
6. **Push** to your fork
7. Create a **Pull Request** to submit

## Important

- Only work in YOUR folder
- Never modify other students' folders
- Use descriptive commit messages
- Push regularly to avoid losing work

## Need Help?

Contact your instructor or raise an issue in this repo.
